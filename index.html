Download for macOS

macOS 14.0 Sonoma 이상 | Apple Silicon & Intel

맥북에서 스크린샷 찍는 방법 알려줘
Command + Shift + 3을 누르면 전체 화면이 캡처되고, Command + Shift + 4를 누르면 원하는 영역만 선택해서 찍을 수 있어요. 파일은 바탕화면에 저장됩니다.
화면 녹화는?
Command + Shift + 5를 누르면 녹화 도구 모음이 나타나요. 전체 화면 또는 선택 영역 녹화를 고를 수 있습니다.
질문을 입력하세요

설치 방법

1

파일 다운로드

위의 Download 버튼을 눌러 Interlude-mini.dmg 파일을 받으세요.
보통 '다운로드' 폴더에 저장됩니다.

2

DMG 파일 열기

다운로드한 .dmg 파일을 더블클릭하면 아래와 같은 창이 열립니다.

Interlude Mini Installer
Interlude-mini
→
Applications
3

앱 아이콘을 Applications로 드래그

왼쪽의 Interlude-mini 아이콘을 마우스로 끌어서 오른쪽 Applications 폴더 위에 놓으세요. 이렇게 하면 설치가 완료됩니다.

끌어다 놓아도 화면에 아무 변화가 없는 것처럼 보일 수 있지만, 정상적으로 복사된 거예요. 그냥 창을 닫으시면 됩니다.
4

앱 실행

Launchpad(런치패드)를 열거나 Finder에서 '응용 프로그램' 폴더로 이동한 후 Interlude-mini를 더블클릭하세요.

처음 실행 시 "확인되지 않은 개발자" 경고가 나올 수 있어요.
그때는 시스템 설정 > 개인정보 보호 및 보안에서 '확인 없이 열기'를 눌러주세요.
5

설치 파일 정리

설치가 끝나면 데스크탑에 남아있는 Interlude Mini Installer 아이콘을 오른쪽 클릭 > "... 추출"을 눌러 제거하세요. 다운로드 폴더의 .dmg 파일도 삭제해도 됩니다.

이 아이콘은 설치용 디스크 이미지가 열려 있는 것이에요. 앱은 이미 Applications에 복사되었으니 안전하게 제거할 수 있습니다.
